Getting Started

To use WhatsApp Web integration:

  1. Create a new session with a unique name
  2. Scan the QR code with your WhatsApp mobile app
  3. Wait for the session to connect
  4. Set up webhooks to receive message notifications
Each session represents a separate WhatsApp account connection.

Status Indicators

Connected - Session is active and receiving messages
Disconnected - Session is inactive
QR Code Ready - Scan with your phone to connect
Connecting - Session is establishing connection
