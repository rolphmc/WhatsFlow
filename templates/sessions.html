Importante para uso com n8n

Os endpoints estão configurados para usar HTTPS, que é o protocolo esperado pelo n8n por padrão.

Se você receber um erro "Client sent an HTTP request to an HTTPS server", você tem duas opções:

  1. Recomendado: Configure um proxy HTTPS para sua aplicação
  2. Alternativa: Configure o n8n para aceitar HTTP em Settings > API > Protocol

Como usar estes endpoints:

Para enviar uma mensagem de texto, faça uma requisição POST com este corpo JSON:

{
  "chatId": "5511999999999@c.us",
  "message": "Olá do n8n!"
}

Para marcar um chat como visto, faça uma requisição POST com este corpo JSON:

{
  "chatId": "5511999999999@c.us"
}

Para iniciar a digitação em um chat, faça uma requisição POST com este corpo JSON:

{
  "chatId": "5511999999999@c.us",
  "duration": 3000
}

Nota: duration é em milissegundos e é opcional (padrão: 3000ms)

Como Começar

Para usar a integração com WhatsApp Web:

  1. Crie uma nova sessão com um nome único
  2. Escaneie o código QR com seu aplicativo WhatsApp móvel
  3. Aguarde a sessão se conectar
  4. Configure webhooks para receber notificações de mensagens
Cada sessão representa uma conexão separada de conta WhatsApp.
